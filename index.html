Alur Proses Distribusi Barang
Diagram berikut menjelaskan langkah demi langkah proses distribusi mulai dari input data hingga penyimpanan ke database.
🖱️ Klik tombol Distribusi
Pengguna memulai proses distribusi dengan menekan tombol Distribusi di aplikasi.
⬇️
📅 Pilih tanggal pemasukan
Tanggal pemasukan dipilih sesuai waktu distribusi yang akan dicatat.
⬇️
📏🌤️⛽ Input detail distribusi
Pengguna memasukkan jarak antar, cuaca hari ini, dan jenis bensin yang digunakan untuk pengiriman.
⬇️
🍎➕ Tambah buah & berat
Klik tombol Tambah Buah untuk memilih jenis buah serta memasukkan berat masing-masing buah.
⬇️
🔁 Tambah buah lain?
Jika ada buah lain, proses input Tambah buah & berat diulangi. Jika tidak, lanjut ke penyimpanan data pemasukan.
⬇️
💾 Simpan pemasukan
Klik tombol Simpan Pemasukan untuk menyimpan data yang telah diinput.
⬇️
🌐 Kirim ke API AI
Data pemasukan yang telah tersimpan dikirimkan ke API untuk diproses oleh sistem AI.
⬇️
🤖 Terima hasil perhitungan AI
Aplikasi menerima hasil kalkulasi dari AI dan menampilkan hasilnya.
⬇️
🗄️ Simpan ke database
Data pemasukan beserta hasil perhitungan AI disimpan ke database untuk keperluan pencatatan dan pelaporan.
⬇️
📋 Arahkan ke daftar pemasukan
Setelah proses selesai, pengguna diarahkan ke halaman Daftar Pemasukan untuk melihat data terbaru.
